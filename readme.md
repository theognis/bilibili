### 一些特点

##### 项目方面

- 基于label简单的推荐算法
- 非关系型数据库的简单使用
- ~~可能是~~ MVC架构
- 使用单独的json文件保存配置，便于修改

##### 安全方面

- 对后端收到的所有参数进行二次检验
- 使用Json Web Token + Refresh Token鉴权，没有使用cookie，避免了csrf攻击
- 资源防盗链
- 数据库层sql语句预处理
- 使用https
- 密码非明文，md5 + 撒盐存储
